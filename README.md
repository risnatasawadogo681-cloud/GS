# Gestion de Stage

Application web développée avec React et Vite dans le cadre d'un projet de stage.

## Fonctionnalités

- Interface de simulation de chat
- Application web réactive et rapide grâce à Vite

## Installation

npm install

## Lancer le projet en local

npm run dev

Le projet sera accessible sur http://localhost:5173

## Construire le projet pour la production

npm run build

## Technologies utilisées

- React
- Vite
- JavaScript (JSX)
